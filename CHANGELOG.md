# Changelog

All notable user-facing changes should be documented here.

This project follows semantic versioning while it is in beta. Breaking changes may still happen before `1.0.0`.

## [Unreleased]

### Added

- Public release documentation for beta testers and contributors.
- Release readiness checklist.
- Security policy.
- Issue templates for bugs, feature requests, and release testing.

## [0.9.0-beta.1] - Unreleased

First planned public beta.

### Expected Scope

- Desktop beta builds for Linux, macOS, and Windows.
- Unsigned Android beta APK artifacts for testing.
- qBittorrent Web UI connection management.
- Torrent list, filtering, detail inspection, and common torrent actions.
- Category, tag, transfer, queue, privacy, Web UI, RSS, and search surfaces where supported by the connected qBittorrent server.
- Local server profile storage with platform secure storage for saved credentials when available.

### Known Limitations

- Builds may be unsigned unless release notes say otherwise.
- Android artifacts are unsigned and intended for testers.
- iOS is not a public distribution target yet.
- Mobile support is less mature than desktop.
- Some qBittorrent features depend on the connected server version and configuration.
