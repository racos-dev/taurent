# Code Of Conduct

Taurent is a small open-source project. Keep discussion direct, technical, and respectful.

## Expected Behavior

- Be clear and constructive.
- Assume good faith when the facts are ambiguous.
- Critique code, design, behavior, and decisions without attacking people.
- Keep bug reports and pull request reviews focused on reproducible facts.
- Respect that maintainers and contributors may have limited time.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or conduct.
- Sharing someone else's private information.
- Posting real credentials, cookies, private tracker URLs, or sensitive logs.
- Repeatedly derailing issues or pull requests after maintainers ask to refocus.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; block users; or decline contributions that violate this code of conduct.

If you need to report conduct privately, contact the maintainer through the same private channel listed for security reports.
